# Cyberpunk Samurai

VS Code themes for Night City — inspired by the comfort of One Dark, with neon accents kept to the chrome so they're easy on the eyes for long sessions.

## Install

- **Marketplace:** search "Cyberpunk Samurai" in the Extensions view.
- **From file:** `code --install-extension cyberpunk-samurai-<version>.vsix`

Then: `Ctrl/Cmd+K Ctrl/Cmd+T` → pick a "Cyberpunk Samurai" theme.

## Recommended settings

```json
{
  "editor.semanticHighlighting.enabled": true,
  "editor.bracketPairColorization.enabled": true
}
```

## Themes

Nine dark themes across five families:

- **Arasaka** — Arasaka Corpo, Arasaka Dim
- **Samurai** — Samurai, Samurai Dim
- **Tyger Claw**
- **Trauma Team** — Trauma Team, Trauma Team Holo *(new)* — petrol-teal medbay, coral-red alerts
- **Maelstrom** — Maelstrom, Maelstrom Chrome *(new)* — blood, rust, and cold chrome
